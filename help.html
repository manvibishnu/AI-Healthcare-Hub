  • How should this application be used?

    This application contains two type of user, one is donor (who is willing to help /donate) and another user is needy (who is suffering from period poverty). Both the users are able to access features of application directly using "User" button. But for donation purpose donor need to register on the site and give required information so that it will be easier for needy to contact them. Features of the application are as follows -

    Map - Map will help user to find donors near their location and get their contact information quickly. Click on the location icon and get required information about donor.

    Flip Card - it is the health tip card which shows user random health tips related to mensuration. Click on "new tip!" Button to get new tip.

    Counselling - for user convince , there is counsellor page where user can schedule meeting with counsellor. User are able to contact our experts (who work as volunteer for helping people) and get advice at no cost.

    Space - space is a global chat where one can connect with like minded people and ask questions , healp others , and discuss the related things.

    Help - this section contains some common queries and answers related to application. It's basically a guide for users and contains common information about application.

    Contact us- user are able to send their feedback directly to us through contact us page.

  • How do I make a sanitary kit donation?

    Donors can register on this application and give right information so that user(needy) are able to contact them easily. once donor register their information will be shown on map . Then user will find donor and contact them directly using provided information.

  • How can I get in touch with the organization/donor who is donating sanitary kits?

    Donor can be found on the map section. Navigate to the map and look for the donor, who is represented by the Tiffany blue colour location icon. Click on that icon to get all of the necessary information about the donor, including contact information.

  • How to help ending period poverty?

    Ending period poverty or period shame cannot begin and finish with simply giving period supplies such as pads and tampons; rather, it must begin with education and addressing discriminatory attitudes and beliefs.

  • Is this application only for women/girls?

    No! This application is for everyone. Menstruation is a biological function; it’s not a “woman thing.” and all the people who menstruate are not women. Genderqueer or non binary people also menstruate. We motivate men to support women and help them get out of this problem.
